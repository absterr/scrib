**Scrib** is a real-time collaborative platform for creating, editing, and organizing documentation and notes across teams.

## Overview

A real-time collaboration tool that lets teams create, edit, and manage documentation and notes together. The application supports simultaneous multi-user editing, ensuring changes are instantly synchronized across all participants. By centralizing shared knowledge and enabling structured collaboration, it helps teams maintain clear, up-to-date documentation and work more efficiently together.

## Features

- **Collaborative editing**: Multiple users can create and edit documents simultaneously with real-time synchronization.
- **Collaborator management**: Invite, remove, and manage collaborators, with control over viewing and editing permissions.
- **AI chat**: A dedicated chat page where users can interact with AI across all notes, or reference a specific note for contextual assistance.

## Running the Project

To run the project locally:

1. Ensure you have Bun installed on your system
2. Clone the repository: `git clone https://github.com/absterr/scrib.git`
3. Navigate to the project root directory: `cd scrib` and install dependencies: `bun install`
4. Configure environment variables in a `.env.local` file.
5. Start the development server: `bun dev` and access the application at `http://localhost:3000`

Scrib is live and it can be accessed at: [Scrib](https://scrib-puce.vercel.app).
